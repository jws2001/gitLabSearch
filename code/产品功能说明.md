# uTools 插件 - GitLab 项目搜索 产品功能说明

https://www.u-tools.cn/docs/developer/basic/getting-started.html 开发者文档

## 一、核心功能

### 1. 项目搜索
- **关键词搜索**：输入项目名、路径或描述关键词实时匹配
- **模糊匹配**：支持拼音首字母、驼峰缩写（如 `gls` 匹配 `gitLabSearch`）
- **搜索范围**：项目名称、项目路径 (path_with_namespace)、项目描述
- **排序策略**：优先显示最近访问 / star 数高 / 最近更新的项目
- **搜索历史**：记录最近搜索词，空查询时展示

### 2. 项目列表展示
- **项目信息**：名称、命名空间、描述、可见性（public/internal/private）
- **图标/头像**：显示项目 avatar，无则显示首字母
- **副标题**：展示所属 Group / 最近更新时间
- **高亮命中**：搜索词在结果中高亮

### 3. 打开方式
- **默认浏览器打开**：回车打开项目主页
- **快捷打开子页面**：
  - `Cmd/Ctrl + Enter` 打开 MR 列表
  - `Alt + Enter` 打开 Issues
  - `Shift + Enter` 打开 Pipelines / CI
  - 其他：Branches、Tags、Wiki、Settings
- **复制链接**：一键复制项目 URL / Clone 地址（HTTPS/SSH）

## 二、配置与认证

### 1. GitLab 连接配置
- **GitLab 地址**：支持公有 gitlab.com 和自建私有实例
- **Personal Access Token (PAT)**：用户填入 token 进行 API 认证
- **Token 权限说明**：提示所需 scope（read_api / read_user）
- **连接测试**：保存前验证 token 有效性
- **多实例支持**（可选）：同时管理多个 GitLab 站点

### 2. 数据同步
- **首次全量拉取**：分页拉取用户可见的所有项目
- **增量更新**：定时（如每小时）拉取最近更新项目
- **手动刷新**：提供"立即同步"按钮
- **本地缓存**：项目列表存入 utools.db，离线可用
- **同步进度提示**：拉取时显示进度条

## 三、uTools 集成

### 1. 插件入口
- **关键词触发**：如 `gl`、`gitlab`、`项目`
- **动态匹配**：`cmds` 中配置 regex，输入即时搜索
- **无头模式**：利用 `list` feature，输入框直接返回结果列表

### 2. 体验优化
- **快捷复制**：`Cmd+C` 复制当前高亮项的链接
- **操作菜单**：右键/快捷键弹出更多操作（MR、Issue、克隆命令等）
- **空状态引导**：未配置时提示用户进入设置页

## 四、设置页

- Token 管理（显示/隐藏、修改、删除）
- GitLab 地址修改
- 同步频率设置
- 打开方式默认行为设置（默认浏览器选择）
- 清除缓存 / 重置插件
- 显示同步状态、项目总数、最后同步时间

## 五、可选增强功能

- **收藏/置顶**：常用项目置顶显示
- **标签分组**：自定义给项目打标签，按标签过滤
- **Group 过滤**：按 Group 筛选
- **MR/Issue 快搜**：扩展搜索 MR 标题、Issue 编号（如 `!123`、`#456`）
- **快速克隆**：调用系统终端执行 `git clone`
- **主题适配**：跟随 uTools 明暗主题

## 六、技术要点

- **GitLab API**：`GET /api/v4/projects?membership=true&per_page=100`，分页处理
- **本地存储**：`utools.db` 存储项目列表与配置
- **图标缓存**：下载 avatar 到本地或用 base64 存储
- **错误处理**：token 失效、网络异常、API 限流的友好提示
- **性能**：大量项目（>1000）时使用 Fuse.js 或自实现索引加速搜索
